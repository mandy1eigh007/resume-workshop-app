# Proof Artifact Templates

> Text the app presents/prints

## Drop-Zone Control Checklist

**Fields:**
* Barricades placed (Y/N)
* Signage visible (Y/N)
* Exclusion radius (ft)
* Spotter assigned (name)
* Start/end time
* Notes
* Verifier (name/sign/date)

## Tool Control Layout

**Fields:**
* Layout photo (no faces)
* Tools labeled (count)
* Missing tools (count)
* Last inventory date/time
* Verifier initials

## Study Log

**Fields:**
* Resource (title)
* Module
* Start/end time
* Score (if applicable)
* Notes
* Signature/date

## Maintenance Log (e.g., chainsaw)

**Fields:**
* Tool ID
* Pre-use inspection (pass/fail)
* Chain tension set (Y/N)
* Fluids checked (Y/N)
* Defects + action
* Date
* Verifier

## Timed Drill Log

**Fields:**
* Drill name
* Distance/weight/target
* Time/result
* Conditions
* PPE
* Verifier
* Date

## Nameplate/Print Read-Through

**Fields:**
* Equipment/print name
* Fields read (e.g., voltage, phase, motor hp)
* Answers
* Verifier
* Date
