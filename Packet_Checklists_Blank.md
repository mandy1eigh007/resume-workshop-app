# Packet Checklists (Blank Templates)

## Week 0–2: Quick Wins

**Goal:** Stack low/no-cost provable signals fast

- [ ] PPE habits documented (checklist with dates)
- [ ] Housekeeping/egress checklist completed (1 site or practice area)
- [ ] Basic tool control layout (photo + checklist, no faces)
- [ ] One timed drill completed (e.g., Timed Carry, Ladder Setup)
- [ ] One awareness sheet signed (e.g., LOTO, Confined Space, Silica)

**Artifacts to attach:**
* Checklist PDFs or scanned forms
* Timed drill log with verifier signature
* Tool control photo (no faces) + checklist
* Awareness sheet with signature and date
