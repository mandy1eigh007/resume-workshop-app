# Cross-Trade Entry Tests & Physicals Appendix

> Summary of buckets and example metrics

## Electrical Aptitude

**What it tests:**
* Arithmetic with fractions/decimals
* Spatial/diagram reasoning
* Reading comprehension

**Metrics:**
* Practice scores over time
* Timed sections
* Signed verification

**Prep resources:**
* Trades Math Drill Set
* Diagram Reasoning Set

## Mechanical Reasoning/Elevator Readiness

**What it tests:**
* Gears, pulleys, sequences
* Schematics

**Metrics:**
* Practice test scores
* Timing
* Signed verification

**Prep resources:**
* Mechanical reasoning practice sets
* Diagram Reasoning Set

## Physical Capability (PCT-style)

**What it tests:**
* Carries, lifts
* Ladder climbs
* Balance tasks

**Metrics:**
* Weights/distances/times
* Pass/fail logs
* Signed verification

**Prep resources:**
* Timed Carry/Lift Plan
* Practice logs

## Ladder/Pole Climb

**What it tests:**
* Height reached
* Rest points
* Tie-in/positioning

**Metrics:**
* Time to height
* Control checks
* Signed verification

**Prep resources:**
* Climb Readiness Practice
* Instructor-signed logs

---

## Verification Options

* Instructor signature
* Supervisor signature
* Designated assessor signature

All verification must include:
* Name
* Date
* Contact information (optional but recommended)
